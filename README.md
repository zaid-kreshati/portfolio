## ⚙️ Local Development

To run this project locally:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/zaid-kreshati/my-portfolio.git
   cd my-portfolio
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   ```

4. **Build for production:**
   ```bash
   npm run build
   ```
